# 「間」Ma - 日式极简博客框架设计文档

## 设计哲学

借鉴日本传统美学中的「間」（Ma，留白）概念。不是空无一物，而是让空间呼吸。每个元素都有存在的理由。

## 视觉系统

### 配色方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景 | `#FAF9F6` | 米白，有温度的白 |
| 主文字 | `#1A1A1A` | 墨黑，柔和不刺眼 |
| 次要文字 | `#6B6B6B` | 灰褐，用于辅助信息 |
| 强调色 | `#E74C3C` | 茜红，仅用于交互状态 |
| 边框/分隔 | `#E8E8E8` | 淡灰，微妙分隔 |

### 排版系统

- **标题字体**: Noto Serif SC（思源宋体）
- **正文字体**: Noto Sans SC（思源黑体）
- **字号层级**: 1.618 黄金比例
  - 正文: 16px
  - H3: 26px
  - H2: 42px
  - H1: 68px

### 间距系统

- 基础单位: 8px
- 文章卡片间距: 64px
- 页面边距: 48px（桌面）/ 24px（移动端）

## 页面结构

### 首页 (index.astro)

**布局:**
- 顶部: 博客名称 + 一句话介绍（左对齐）
- 主体: 文章列表（垂直排列，大留白）
- 侧边: 标签云（不规则排列）

**文章卡片结构:**
- 标题（H2）
- 发布日期 + 阅读时间
- 摘要（2-3行）
- 分类标签

### 分类页 (categories.astro)

**布局:**
- 横向滚动分类标签
- 选中分类高亮显示
- 该分类下的文章列表

### 标签页 (tags.astro)

**布局:**
- 标签云，大小根据文章数量变化
- 不规则网格布局

### 归档页 (archive.astro)

**布局:**
- 时间轴，垂直排列
- 月份分组，简洁清晰

## 交互细节

- **悬停**: 文字微微右移 4px
- **点击**: 淡入淡出，无弹跳
- **滚动**: 导航栏渐隐渐现

## 数据结构

### 文章 Frontmatter

```yaml
---
title: 文章标题
date: 2026-03-13
category: 分类名
tags: [标签1, 标签2]
description: 文章摘要
readingTime: 5分钟
---
```

## 组件清单

1. **Header** - 顶部导航
2. **Footer** - 底部信息
3. **PostCard** - 文章卡片
4. **TagCloud** - 标签云
5. **CategoryNav** - 分类导航
6. **ArchiveTimeline** - 归档时间轴
